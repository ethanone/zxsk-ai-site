# 🤖 AI 定制化指南 - 如何用提示词快速为新公司搭建网站

本指南教您如何使用 AI 助手（如 Claude、ChatGPT、Cursor）快速将此模板定制为任何公司的官网。

---

## 📋 准备工作

### 1. 收集客户信息

在开始之前，收集以下信息：

```
✅ 公司基本信息
  - 公司名称（中英文）
  - 成立时间
  - 公司简介（一句话）
  - 详细介绍（2-3段）
  - 联系方式（邮箱、电话、地址）

✅ 品牌元素
  - Logo 文件
  - 品牌色（主色、辅色、强调色）
  - 字体偏好（如有）

✅ 业务信息
  - 核心服务/产品（3-4个）
  - 目标客户群体
  - 核心优势（4个）
  - 客户案例/见证（2-3个）

✅ 其他
  - 网站风格偏好（科技感、简约、专业等）
  - 参考网站（如有）
```

## 🎯 核心提示词模板

### 提示词 1: 初始化项目

```

公司信息：
- 公司名称：[中文名] / [English Name]
- 行业：[行业]
- 成立时间：[年份]
- 核心业务：[业务描述]
- 品牌色：主色 #XXXXXX，辅色 #XXXXXX，强调色 #XXXXXX

请帮我：
1. 更新 src/data/companyData.json 和 companyData.en.json 中的所有公司信息
2. 将品牌色应用到 src/app/globals.css
3. 更新 package.json 中的项目名称和描述
4. 更新 src/app/layout.tsx 中的 SEO 元数据

保持现有的技术栈和代码结构不变，只替换数据和配置。
```

```

目标关键词：
- [关键词1]

- [关键词3]

Open Graph 信息：
- og:title: [标题]
```

### 提示词 6: 样式调整

```
请根据 [公司名称] 的品牌风格调整网站样式：

品牌色：
- 主色（Primary）：#XXXXXX - 用于标题、按钮
- 辅色（Secondary）：#XXXXXX - 用于副标题、图标
- 强调色（Accent）：#XXXXXX - 用于 CTA 按钮

设计风格：[科技感/简约/专业/活力] 

具体调整：
1. 更新 src/app/globals.css 中的 CSS 变量
2. 调整按钮样式（.cta-button, .secondary-button）
3. 如果是科技感，增强卡片阴影和渐变效果
4. 如果是简约风，减少动画和装饰元素

保持响应式布局和可访问性标准。
```

## 💡 提示词技巧

### ✅ 好的提示词

```
✓ 具体明确："将主色改为 #0066CC"
✓ 提供上下文："这是一家 AI SaaS 公司"
✓ 分步执行："先更新 JSON，再更新样式"
✓ 给出示例："标语类似'用 AI 驱动创新'"
```

### ❌ 避免的提示词

```
✗ 过于笼统："让网站更好看"
✗ 多任务混合："改颜色、加功能、优化性能"
✗ 缺少细节："更新公司信息"（没说是哪家公司）
✗ 技术假设："用那个库实现"（没说哪个库）
```
